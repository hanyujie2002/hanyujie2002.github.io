How to use AppImage

What is AppImage?

AppImage is a kind of portable execute file for Linux. AppImage features running in all Linux distributions, being easy to install, easy to run.

How to get AppImage?

It's easy to get as many programmers may provide their AppImage edition of their software in the release in their GitHub.

How to use AppImage?

For example, a new AppImage named example.appimage was downloaded in your ~/ folder.

cd ~
chmod +x ./example.appimage
./eample.appimage

This app is launched then.
